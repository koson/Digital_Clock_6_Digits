

## การประกอบวงจรนับเวลา (Counter)

ในส่วนนี้เราจะมาดูขั้นตอนการประกอบวงจรนับเวลา (Counter) ซึ่งเป็นส่วนที่สำคัญมาก ๆ ในโครงงานนี้ เพราะเป็นส่วนที่ทำหน้าที่นับและจัดเก็บค่าเวลา (ชั่วโมง นาที วินาที) ในรูปแบบ BCD ซึ่งจะถูกส่งไปยังส่วนแสดงผล (View) เพื่อแสดงผลออกมาให้เราเห็น

***ถ้าหากมีข้อสงสัยในการทำงานของวงจรนับเวลา (Counter) สามารถดูได้จากเอกสารนี้  [Counter Design](./Documents/Counter-Design.md) ซึ่งจะอธิบายการทำงานของวงจรนับเวลาอย่างละเอียด และมีตัวอย่างการคำนวณและการออกแบบวงจรนับเวลาที่ใช้ในโครงงานนี้ด้วย***

นักศึกษาสามารถใช้วงจรในหน้าที่ 3 ของไฟล์ใน Kicad ที่ชื่อว่า "Counter" ซึ่งเป็นวงจรนับเวลาที่ออกแบบมาแล้วในรูปแบบ BCD โดยมีการใช้ไอซี 74LS90 ในการนับและจัดเก็บค่าเวลา และมีการเชื่อมต่อกับส่วนแสดงผล (Display) ผ่านทางขั้วต่อ (Connector) เพื่อให้สามารถส่งสัญญาณไปยังส่วนแสดงผลได้อย่างถูกต้องและง่ายดาย

ถ้ามีข้อสงสัยในการประกอบวงจรนับเวลา (Counter) สามารถดูได้จากเอกสารนี้ [Construction Process of Counter](./Documents/Construction-Process-Counter.md) ซึ่งจะอธิบายขั้นตอนการประกอบวงจรนับเวลาอย่างละเอียด และมีภาพประกอบเพื่อให้เข้าใจได้ง่ายขึ้นด้วยเช่นกัน หรือไม่ก็ให้เปิด issue ถามมาใน github repo ได้เลยครับ
